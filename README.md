# Bitlink

## Описание работы скрипта:
___
Скрипт сокращает ссылку через сервис bit.ly, а при введении сокращенной ссылки выводит количество переходов по ней.

## Зависимости
___
Python3:

```
 pip3 install python-dotenv
 pip3 install requests     
```
